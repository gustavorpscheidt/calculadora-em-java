Este foi um projeto acadêmico feito durante uma aula de programação no Instituto Federal Catarinense (IFC) Campus São Bento do Sul com o objetivo de testar as habilidades de utilizar a GUI.

O projeto consiste basicamente em Criar uma calculadora com as funções básicas de soma,subtração,multiplicação, divisão, inversão, fatorial e potenciação, além de constar com um histórico dos calculos feitos.
